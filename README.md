# 💇‍♀️ Diamond Style Braids – Premium Hair Braiding Platform

A modern, responsive, and full-stack web application designed for a premium hair braiding salon. The platform showcases luxury braiding services, catagories styles with custom tabs, features a dynamic home hero slider, and lets users view individual service details seamlessly.

## ✨ Features

### 💻 Frontend Features
* **Premium Hero Carousel:** An automated, ultra-responsive background image slider with continuous animations, dynamic text glow (`drop-shadow`), and a luxury glassmorphism center console.
* **Dynamic Services Swiper:** Fully responsive slider powered by `Swiper.js` with functional `lucide-react` navigation arrows.
* **Category Tabs Switching:** Interactive filter system allowing users to instantly switch between "Classic Braid Styles" and "Finishing/Maintenance Services".
* **Service Detail Pages:** Dynamic routing setup (`/service/:id`) fetching real-time data from the backend to present descriptive, high-quality style overviews.

### ⚙️ Backend Features
* **RESTful API Structure:** Secure routes built with Express.js to fetch all services collectively or fetch unique styles via dynamic database parameter matching.
* **MongoDB Integration:** Seamless object storage querying and data retrieval using native `ObjectId` structures.
* **Robust Global Cors Configuration:** Fully optimized middleware preventing domain blockages across cross-origin deployments.

## 🛠️ Tech Stack

### Frontend
* **Core:** React.js, React Router DOM (v6)
* **Styling:** Tailwind CSS
* **Components & Icons:** Swiper.js, Lucide React

### Backend & Database
* **Runtime Environment:** Node.js, Express.js
* **Database:** MongoDB Atlas (Native MongoDB Driver)
* **Environment Management:** Dotenv
